## Sublime
To do this Action                   | You must take these Steps
------------------------------------|--------------------------
Align selected lines                | Ctrl + Cmd + a

##### Good SNIPPETS:

**IN a C file:**

prefix      | result
------------|--
`def`       |  `define`
`do`        |  `do/while`
`enum`      |  `enum` declaration
`for`       |  really nice!
`fund`      |  function declaration
`if`        |  `if`
`inc`       |  `include <>`
`Inc`       |  `include ""`
`main`      |  `main`
`pr`        |  `printf`
`td`        |  `type` definition
`tds`       |  `struct` declaration
`wh`        |  `while`

**IN a PYTHON file:**

prefix          |  result
----------------|--
`cl`            |  class declaration
`def`           |  function
`deff`          |  simpler version
`for`           |  for
`imp`           |  import
`property`      |  property
`try`           |  buncha options given
`wh`            |  while
